# CollabU - 学生团队协作与项目管理系统

## 项目状态

所有核心功能已根据设计文档 (`DESIGN.md`) 实现完成。

### 已完成功能清单

1.  **用户系统**
    -   注册 (支持学号)、登录 (JWT)、退出。
    -   前端页面汉化完成。

2.  **团队管理**
    -   团队列表展示。
    -   创建团队、加入团队 (邀请码机制)。
    -   团队详情页：查看成员、邀请码、项目列表。
    -   退出团队。

3.  **项目管理**
    -   在团队内创建项目 (设置起止时间)。
    -   项目详情页：任务看板 (Kanban) 与 甘特图 (Gantt) 切换。
    -   项目状态可视化。

4.  **任务管理**
    -   看板视图：待处理、进行中、已完成。
    -   任务创建、编辑、状态流转。
    -   **任务详情抽屉**：
        -   **子任务**：添加、勾选完成。
        -   **评论区**：发表和查看评论。
        -   **实时聊天**：基于 WebSocket 的任务内即时通讯。
        -   **文件共享**：上传、下载、删除文件。

5.  **通知中心**
    -   首页集成通知铃铛图标。
    -   查看通知列表、标记已读。

### 运行指南

**后端**:
```bash
cd backend
$env:FLASK_APP="run.py"; flask run --port 5000
```

**前端**:
```bash
cd frontend
npm run dev
```

### 技术栈
-   **后端**: Flask, SQLAlchemy, SocketIO, JWT
-   **前端**: Vue 3, Vite, Element Plus, Pinia, Axios, Socket.io-client, DHTMLX Gantt
